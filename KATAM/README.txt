Katam.ipynb is a code for finding rainfall in each region.
NetCDF4 module is used to open satellite data files and Pandas to process them.

Step:
1. Look for Latitude and Longitude from each region. (Based on Google Maps)
2. Look for a grid where the area is located.
3. Dividing for wet years, normal years, and dry years according to ENSO information.
4. Extracting rainfall data according to each region.
5. Saving data in the form of Excel.
